# Study project with Vue.js 2

## Packages
- [Vue CLI](https://cli.vuejs.org/)
- [Vue Router](https://router.vuejs.org/)
- [Vuetify](https://vuetifyjs.com/)
- [Vuex](https://vuex.vuejs.org/)
- [Axios](https://axios-http.com/)

## Project setup
```
npm install
```
